# 基础使用

学习 Plan Viewer 的基本操作。

## 界面概览

Plan Viewer 的主界面包含以下区域：

```
┌─────────────────────────────────────────────────────────────┐
│  ┌─────────────┐  ┌─────────────────────────────────────┐   │
│  │             │  │                                     │   │
│  │  计划列表   │  │           计划内容视图              │   │
│  │             │  │                                     │   │
│  │  - Plan 1   │  │   Markdown 渲染 + Mermaid 图表      │   │
│  │  - Plan 2   │  │                                     │   │
│  │  - Plan 3   │  │                                     │   │
│  │             │  │                                     │   │
│  └─────────────┘  └─────────────────────────────────────┘   │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                    评论侧边栏                        │   │
│  │  - 评论 1                                            │   │
│  │  - 评论 2                                            │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

## 计划列表

左侧边栏显示所有可用的计划文件：

- 文件名作为标题
- 最后修改时间
- 点击选择查看

## 计划视图

主视图区域显示计划内容：

- Markdown 渲染
- Mermaid 图表实时渲染
- 代码语法高亮
- 目录导航

## 添加评论

### 章节评论

1. 找到要评论的章节
2. 点击章节标题旁的 `+` 按钮
3. 在弹出的对话框中输入评论
4. 点击保存

### 内联评论

1. 选择要评论的文本
2. 松开鼠标后会出现评论选项
3. 输入评论内容
4. 点击保存

## 主题切换

点击右上角的主题切换按钮：

- 🌙 深色主题
- ☀️ 浅色主题

## 快捷操作

| 操作 | 说明 |
|------|------|
| 点击计划 | 打开计划 |
| 点击 `+` | 添加章节评论 |
| 选择文本 | 添加内联评论 |
| 点击评论 | 跳转到相关位置 |

## 文件位置

计划文件存储在：

```
~/.claude/plans/
├── plan-001.md
├── plan-002.md
└── ...
```

你可以直接编辑这些文件，Plan Viewer 会自动检测变更并更新显示。
