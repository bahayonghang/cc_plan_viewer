# 审查工作流

使用 Plan Viewer 审查 Claude Code 计划的分步指南。

## 1. 打开计划

点击活动栏中的 **Plan Viewer** 图标，然后点击要审查的计划。Webview 会以渲染后的 Markdown 格式打开。

## 2. 阅读计划

滚动浏览渲染后的内容。计划按标题分为多个章节，Mermaid 图表内联渲染。

如需同时查看原始文件，点击工具栏的 **Editor** 按钮即可在旁边打开 `.md` 文件。

## 3. 添加评论

### 在章节上评论

点击任意章节标题右侧出现的 **`+`** 按钮，评论表单会在标题下方展开。

### 在选中文字上评论

1. 在 Markdown 面板中选中任意文字
2. 选中文字上方出现 **💬 Comment** 提示
3. 点击提示打开评论表单——选中文字将作为上下文预填

### 填写表单

```
┌─ 评论表单 ──────────────────────────────────────────────────────┐
│  💬 评论于："此处为选中文字" [第 42 行]                          │
│                                                                │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │ 在此输入评论…                                             │  │
│  └─────────────────────────────────────────────────────────┘  │
│                                                                │
│  类型：[💬 注释 ▾]                        [取消] [提交]          │
└────────────────────────────────────────────────────────────────┘
```

**评论类型：**

| 类型 | 表情 | 使用场景 |
|---|---|---|
| 注释 | 💬 | 一般性观察或备注 |
| 建议 | 💡 | 提出具体修改或改进意见 |
| 问题 | ❓ | 向计划作者寻求说明 |
| 批准 | ✅ | 明确批准该章节 |
| 拒绝 | ❌ | 标记该章节需要修改 |

**键盘快捷键：**

| 快捷键 | 操作 |
|---|---|
| `Ctrl+Enter` / `Cmd+Enter` | 提交评论 |
| `Esc` | 取消并关闭表单 |

## 4. 浏览评论

点击工具栏中的 **Comments** 按钮或 **💬 N** 徽章打开 **评论面板**。

面板按章节标题分组显示评论，点击任意评论可将 Markdown 面板滚动到对应章节。

Markdown 面板中每个章节的 `+` 按钮上也会显示内联评论计数徽章。

## 5. 删除评论

- **从评论面板：** 点击任意评论卡片上的 🗑 图标
- **从 Markdown 面板：** 点击章节内联评论卡片上的 🗑 图标

删除后会弹出 Toast 通知确认。

## 6. 评论持久化

评论存储在 VS Code 的 `globalState` 中，编辑器重启后仍然保留。若启用了 `planViewer.embedCommentsInMarkdown`（默认开启），评论也会写入计划的 `.md` 文件，位于底部的 `## 📝 Review Comments` 章节下。

嵌入的评论格式如下：

```markdown
## 📝 Review Comments

### 💡 SUGGESTION (on: "建议的方案") [第 15 行]

> 考虑将其拆分为更小的子任务，便于跟踪进度。

_— Reviewer, 2024/06/15 10:30_

---

### ❓ QUESTION [第 28 行]

> 同步步骤的预期延迟是多少？

_— Reviewer, 2024/06/15 10:32_
```

这使得评论具有**可移植性**——它们随计划文件一同传递。

## 技巧

- 使用 **批准**（✅）和 **拒绝**（❌）评论向计划作者传达审查状态
- 使用 **建议**（💡）让 Claude Code 修改特定章节
- 当计划有多个章节分散着评论时，**评论面板** 是绝佳的概览工具
- 如果你经常审查，可以在侧边栏关闭 **分组**，按时间顺序查看所有计划
