# 快速开始

五分钟内上手 Plan Viewer。

## 前提条件

- **Visual Studio Code** ≥ 1.85.0
- **Claude Code**（或任何会将计划 `.md` 文件写入 `~/.claude/plans` 的工具）

## 安装扩展

1. 打开 VS Code
2. 打开扩展面板（`Ctrl+Shift+X` / `Cmd+Shift+X`）
3. 搜索 **"Plan Viewer"**
4. 点击 **安装**

::: tip 其他安装方式
也可以在浏览器中通过 [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=anthropic-community.plan-viewer) 安装，或从 `.vsix` 文件安装。详见 [安装](./installation)。
:::

## 打开第一个计划

1. 点击活动栏（左侧边栏）中的 **Plan Viewer 图标**
2. **Plans** 面板显示 `~/.claude/plans` 中的所有 `.md` 文件
3. 点击任意计划条目，即可打开预览 Webview

完成！你现在可以浏览和审查 Claude Code 计划了。
