# 基本使用

## 界面概览

Plan Viewer 在 VS Code 活动栏中添加了一个专属图标。点击它可打开 **Plans** 侧边栏。

```
┌─ 活动栏 ────────────────────────────────────────────────────────┐
│  [📋]  ← Plan Viewer 图标                                        │
└──────────────────────────────────────────────────────────────────┘

┌─ Plans 侧边栏 ───────────┐  ┌─ Webview 面板 ────────────────────────┐
│  PLANS              [↺] │  │  ┌─ 工具栏 ──────────────────────┐  │
│                         │  │  │  plan-name.md   💬 3  [Editor] │  │
│  ▼ my-project           │  │  └─────────────────────────────────┘  │
│    ├─ plan-2024.md      │  │                                        │
│    └─ plan-draft.md  ←──┼──┼──（点击打开）                          │
│                         │  │  # 计划标题                            │
│  ▼ other-project        │  │                                        │
│    └─ feature.md        │  │  章节内容…                             │
│                         │  │                                [+ 💬]  │
│  （未分组）             │  │  ┌─ 评论卡片 ──────────────────────┐  │
│    └─ misc.md           │  │  │ 💬 "看起来不错"         🗑         │  │
│                         │  │  └────────────────────────────────┘  │
└─────────────────────────┘  └────────────────────────────────────────┘
```

## Plans 侧边栏

### 工具栏操作

| 图标 | 命令 | 说明 |
|---|---|---|
| `↺` | 刷新计划 | 手动刷新计划列表 |
| `⚙` | 设置 | 打开筛选到 `planViewer.*` 的 VS Code 设置 |
| `🌲` | 切换分组 | 在分组/平铺计划列表之间切换 |
| `⊞` | 展开全部 | 展开所有项目分组 |
| `⊟` | 折叠全部 | 折叠所有项目分组 |

### 项目分组

计划会根据从计划内容中提取的项目名称自动分组。提取规则按优先级依次为：

1. 开头的 `cwd:`、`Working directory:` 或 `Project:` 元数据行 → 使用路径的 basename
2. 内容中的绝对路径 → 使用 basename
3. `# 标题` 行 → 使用 `:`/`-`/`–` 之前的文字（最多 20 个字符）

无法识别项目的计划显示在 **（未分组）** 下。点击工具栏的 `🌲` 按钮可切换为平铺列表。

### 右键菜单

右键点击计划条目，可在 VS Code 内置文本编辑器中直接打开该 `.md` 文件。

## Webview 面板

点击侧边栏中的任意计划，即可在 Webview 面板中打开。面板分为三个部分：

### 工具栏

- **计划文件名** — 显示在左侧
- **💬 N** — 显示总评论数的徽章（点击可切换评论面板）
- **Editor** 按钮 — 在 VS Code 内置文本编辑器中打开 `.md` 文件

### Markdown 面板

主内容区域以样式化 Markdown 渲染计划，包含：

- 带语法高亮的代码块
- 内联渲染的 Mermaid 图表
- 章节级别的评论触发器

每个章节标题右侧都有一个 **`+`** 按钮。点击可为该章节打开评论表单。若章节已有评论，按钮会显示评论数而非 `+`。

### 悬停评论

在 Markdown 面板中选中任意文字，选中文字上方会出现 **💬 Comment** 提示。点击即可打开预填了选中文字的评论表单。

### 评论面板

点击工具栏的 **Comments** 按钮打开右侧可折叠面板，所有评论按章节分组列出。点击任意评论可滚动到对应章节。

## 配置项

打开 VS Code 设置（`Ctrl+,`）并搜索 `planViewer` 查看所有选项：

| 设置 | 默认值 | 说明 |
|---|---|---|
| `planViewer.plansDirectory` | `~/.claude/plans` | 自定义计划目录路径 |
| `planViewer.fontSize` | `14` | 计划内容字体大小（px） |
| `planViewer.lineHeight` | `1.7` | 计划内容行高 |
| `planViewer.embedCommentsInMarkdown` | `true` | 将评论嵌入 `.md` 文件 |
| `planViewer.groupByProject` | `true` | 在侧边栏按项目分组 |
