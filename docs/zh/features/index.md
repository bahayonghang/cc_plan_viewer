# 功能概览

Plan Viewer 专为在 VS Code 内审查 Claude Code 计划文件而生。以下是它的全部功能。

## 功能一览

| 功能 | 说明 |
|---|---|
| [评论系统](./comments) | 为章节或选中文字添加 5 种评论类型；双写存储 |
| [Mermaid 图表](./mermaid) | 内联渲染 Mermaid 图表，自动适配主题颜色 |
| [文件监听](./file-watcher) | 磁盘文件变化时自动刷新计划列表和预览 |
| [主题](./themes) | 跟随 VS Code 配色主题；支持自定义字体和行高 |

## 快速概览

### 计划发现
- 从 `~/.claude/plans` 读取 `.md` 文件（可配置）
- 从计划内容提取项目名称，自动分组
- 自动提取简化的子项名称并为其分配动态的彩色大图标渲染
- 按修改时间排序，最新在前
- 通过下拉菜单直接热切换 OpenSpec ↔ Claude Plans 不同视图

### 内联审查
- 章节级别和文字选择评论触发器
- 评论面板一次性浏览所有评论
- 键盘快捷键：`Ctrl+Enter` 提交，`Esc` 取消

### 持久化
- 主存储：VS Code `globalState`（重启后保留）
- 可选副存储：直接嵌入 `.md` 文件本身
- 每次加载计划时双向同步，两个存储来源不会产生分歧

### 渲染
- 使用 `marked` 完整渲染 Markdown
- 使用 `highlight.js` 语法高亮
- 支持 Mermaid 图表（懒加载）
- 遵循 VS Code 深色/浅色主题
